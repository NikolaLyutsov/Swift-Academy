  • Content/Съдържание
  • Padding/Вътрешно отстояние
  • Border/Граница
  • Margin/Външно отстояние между елементите

Стандартната стойност на property box-sizing: content-box.
В този случай зададените height & width се отнасят за само за съдържанието. padding и border се добавят към тези размери.

При box-sizing: border-box padding и border са включени в зададените размери

При каквата и да е зададена стойност на box-sizing margin не се включва зададения размер, той е отстояние между елементите
